# レイアウト再構成の実装計画

分析結果が見づらいという課題を解決するため、画面上部を分割し、シミュレーターと分析結果を並列に表示するレイアウトに更新します。

## 変更内容

### UI構成の変更 (`src/features/serve/ServeScene.tsx`)

画面レイアウトを以下のように変更します：

- **これまでのレイアウト**:
  - [上部] シミュレーター (全幅)
  - [下部] コントロールパネル + 分析結果 (スクロールエリア)

- **新しいレイアウト**:
  - [上部] (高さ 100vh - 下部パネル高さ)
    - [左 2/3] シミュレーター
    - [右 1/3] 分析結果パネル (スクロール可能)
  - [下部] コントロールパネル (固定高さ 200px 程度、スクロールエリア)

#### 修正箇所 [MODIFY] [ServeScene.tsx](file:///c:/Users/hoge/Documents/GitHub/TennisServeSimulator/src/features/serve/ServeScene.tsx)

1.  **コンテナの分割**:
    - `containerRef` を含む div を、横並びの Flexbox コンテナで包みます。
    - シミュレーター部分に `flex: 2`、分析結果パネル部分に `flex: 1` を割り当てます。
2.  **`AnalysisPanel` の移動**:
    - `AnalysisPanel` を下部エリアから上部右側のコンテナへ移動します。
    - 右側パネルには `overflowY: 'auto'` を設定し、内容が多い場合でもスクロールできるようにします。
