# 高度な人型モデルとテニスモーションの統合完了

サーバーとレシーバーのモーションを大幅に強化し、テニスの「しなり」や「躍動感」を感じられる高度なプロシージャルアニメーションを実装しました。

## 主な改良点

### 1. 骨格構造の拡張
関節を増加させ、全身の連動（キネティック・チェーン）を表現可能にしました。
- **脊椎 (Spine)**: サーブ時の「背中の反り」やスイング時の「捻り」を実現。
- **膝 (Knees)**: サーブのタメ（屈伸）やレシーブの低い構え（Ready Pose）を追加。
- **手首 (Wrists)**: インパクト時のラケットの走りを表現。

### 2. 現実味のあるテニスモーション
- **サーブ (Trohpy Pose & Timing)**: 
  - **通常サーブ（打点2m以上）**: トスを上げてからインパクトするまで0.6秒。膝を曲げ、身体を弓なりに反らせる「タメ」の動作から爆発的にインパクトします。
  - **アンダーサーブ（打点2m未満）**: トスを省略し、0.2秒で素早くインパクト。意表を突くショットにより、相手の準備時間を削る効果を視覚的に再現しました。
- **レシーブ & 移動**:
  - **サイドシャッフル**: 横方向への移動時に、正対したままステップを踏むテニス特有の動きを導入。
  - **レディポーズ**: 移動待機中に重心を低く保つ戦闘態勢を追加。
- **スイングの深化**: 肩の入れ替えと腰の回転を同期させ、力強いスイングを視覚化。

### 3. 同期精度の向上
ボールの軌道・速度とキャラクターのモーションをより密接に同期させました。
- **トスとインパクトの同期**: アニメーション開始直後にボールが飛んでいくのではなく、適切なタイミング（インパクト）に合わせて物理シミュレーションによる飛行が開始されるように調整しました。
